📊 Amazon Sales Analysis – Power BI Dashboard

An interactive 2-Page Power BI Dashboard designed to analyze Amazon’s sales performance, profitability, customer behavior, and discount impact.

This project provides both executive-level insights and deep analytical exploration to support business decision-making.

📌 Project Overview

Amazon operates across multiple product categories, regions, and payment channels. This dashboard consolidates fragmented data into a unified analytical solution to:

Monitor overall sales performance

Evaluate profitability trends

Analyze customer behavior

Measure discount effectiveness

Identify regional performance drivers

🎯 Objectives

Build a 2-Page Interactive Power BI Dashboard

Enable high-level monitoring for leadership

Provide deep-dive analysis for operational decisions

Deliver actionable business insights

📂 Dataset Description

The dataset contains transaction-level data including:

Order details & dates

Product categories

Pricing & discount information

Revenue & profit metrics

Customer regions

Payment methods

Customer ratings & review counts

Profit Trend Over Time

🔹 Performance Analysis

Revenue by Product Category

🟩 Page 2 — Deep Analysis (WHY it is happening)
🔹 Profitability Analysis

Profit by Category

Discount % vs Revenue

Discount % vs Profit

Category Performance by Region

Cross-filtering between visuals

Drill-down (Time hierarchy)

Drill-through navigation between pages

🎨 Design Principles

Clean corporate theme

Consistent color palette

High readability

Minimal visual clutter

Logical layout structure

📈 Key Insights

High revenue does not always mean high profit.

Customer ratings and reviews influence sales performance.

Excessive discounts can reduce profitability.

Regional preferences impact category performance.

Balanced pricing strategy is essential for sustainable growth.

🏁 Conclusion

This 2-page interactive dashboard provides a comprehensive analytical view of Amazon’s sales operations. It enables stakeholders to move from monitoring performance to identifying root causes and making data-driven decisions effectively.
